Steganography Tool

Un programa para ocultar mensajes dentro de imágenes o audios, demostrando cómo funciona la esteganografía en la ciberseguridad.

La herramienta toma un archivo de entrada (imagen o audio) y un mensaje que el usuario desea ocultar. Utiliza técnicas de esteganografía para insertar el mensaje en bits menos significativos del archivo, lo que lo hace casi imperceptible al ojo humano o al oído. También incluye una opción para extraer mensajes ocultos si el usuario tiene la clave o método correcto. Este proyecto puede incluir una interfaz gráfica simple para cargar archivos y mostrar resultados.
