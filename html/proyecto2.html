Password Strength Checker

Una herramienta interactiva que evalúa la fortaleza de las contraseñas. Los usuarios pueden ingresar contraseñas y obtener puntuaciones basadas en longitud, complejidad y patrones comunes.

Este proyecto consiste en una aplicación que evalúa las contraseñas en tiempo real. Los usuarios escriben su contraseña, y la herramienta la analiza en base a diferentes criterios: longitud, uso de caracteres especiales, números, mayúsculas y minúsculas, y la detección de patrones comunes. También puede integrarse con bases de datos de contraseñas filtradas para avisar si la contraseña ha sido comprometida. El resultado se muestra como una barra de progreso o un mensaje categorizando la seguridad (débil, moderada, fuerte).
