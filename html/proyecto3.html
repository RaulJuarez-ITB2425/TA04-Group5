Basic Vulnerability Scanner

Un proyecto que analiza sitios web en busca de vulnerabilidades básicas como enlaces rotos, headers inseguros o puertos abiertos.

El escáner analiza un sitio web o sistema en busca de vulnerabilidades comunes. Esto incluye verificar configuraciones inseguras, puertos abiertos, encabezados HTTP inseguros y la exposición de información sensible. Utiliza scripts básicos y herramientas de automatización para simular ataques comunes. Al final, genera un informe con los problemas encontrados y recomendaciones para solucionarlos.
